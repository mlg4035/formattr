## AppHaven Formatr
### From rough draft to polished, export-ready documents in minutes

**AppHaven Formatr** is your AI-powered writing and document production assistant — built to turn messy text into clean, structured output you can actually ship.

---

### What it does

- **Polishes raw text fast**
  Paste notes, transcripts, drafts, or long-form content and get a clean, readable result in seconds.

- **Dual AI provider support**
  Choose **Google Gemini** (default) or **OpenAI** based on your preference.

- **Controls rewrite intensity**
  Pick the level of editing you want:
  - Strict (structure-focused)
  - Minimal (light grammar/punctuation)
  - Thorough (clarity and flow)

- **Advanced AI formatting controls**
  Decide how aggressively the model should use:
  - Bullets/lists
  - Pull quotes
  - Numbered steps
  - Section summaries
  - Tables
  - Callouts
  using `off / auto / prefer` modes.

- **Built-in + custom presets**
  Use one-click presets (Article, Executive Brief, Tutorial, etc.) or save your own reusable presets.

---

### Output quality + transparency

- **LLM-generated title**, with **manual editing** and save-to-history support
- **Diff view** to see exactly what changed
- **One-click copy**
- **Persistent user settings** so your preferred formatting/export behavior is remembered

---

### Exports built for real workflows

- **Markdown export (`.md`)** with title-aware output
- **Word export (`.docx`)** with:
  - true Word tables (not raw markdown)
  - markdown alignment mapping
  - finance-aware numeric column auto-alignment
  - robust heading/list/quote handling across template differences

---

### Word template + styling controls

- Optional **Use project template** mode (e.g., `templates/base_template.docx`)
- Template discovery dropdown + manual path fallback
- Template health diagnostics:
  - heading-style detection
  - paragraph/all style inspection
  - browser-safe plain-text fallback view
- Optional H1/H2/H3 **font + size overrides**
- Reset heading overrides in one click
- Graceful fallback behavior when template styles are missing or renamed
