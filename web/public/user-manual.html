Formattr User Guide

Browser-based text formatting with your own API keys. Keys and content stay client-side unless you send them to OpenRouter or ToolerBox.

Quick start

  1. Enter your OpenRouter API key (required).
  2. Click Load OpenRouter Models and pick a model.
  3. Paste your text into Input, or load a YouTube transcript.
  4. Adjust formatting options or choose a Preset.
  5. Click Format.
  6. Copy, download Markdown, or export Word from Output.

API keys

OpenRouter API key (required)

Used for model listing and formatting. Get a key at openrouter.ai. You pay OpenRouter directly; Formattr does not bill you.

  • Enter the key in the password field.
  • Click Load OpenRouter Models before formatting.

ToolerBox API key (optional)

Required only for Load Transcript on YouTube videos.

Remember API keys

Option Behavior
Don't remember (default) Keys exist only in memory until you refresh or leave the page.
This browser session Saved in sessionStorage. Survives refresh; cleared when the browser closes.
On this device Saved in localStorage. Persists between visits on the same browser.

Keys are never stored on Formattr servers. Use Clear saved keys to wipe stored keys and reset the preference.

Model and prompt settings

Text changes

Setting What it does
Strict (structure only) Preserves wording; fixes structure and basic Markdown organization.
Minimal (spelling & punctuation) Fixes spelling, punctuation, and small grammar issues.
Thorough (grammar & clarity) Improves grammar, clarity, and flow while keeping the original intent.

The model is instructed not to invent facts, quotes, or citations.

Markdown toggles

Control which Markdown elements the model may use: bold, italics, and H1/H2/H3 headings. Unchecked options are forbidden.

Structure controls

Each structure type supports three modes:

  • Off — do not use.
  • Auto — use when naturally helpful.
  • Prefer — prefer using when readability improves.

Structure options: Lists, Pull quotes, Numbered steps, Section summaries, Tables, and Callouts.

Presets

Presets bundle text-change level and formatting toggles into one-click configurations.

Preset Best for Summary
Minimal Cleanup Light touch-ups Minimal text changes; basic structure only.
Article Long-form writing Thorough editing; prefers lists and pull quotes.
Executive Brief Business summaries Minimal changes; prefers lists, summaries, tables, callouts.
Tutorial How-to content Thorough editing; prefers steps, lists, tables, callouts.

Custom presets: configure settings, enter a name, click Save Preset. Saved presets persist in this browser. Select one and click Delete Preset to remove it.

Show Diff View displays side-by-side original and formatted text below the output.

Input

  • Paste or type source content into the text area.
  • For YouTube: enter the 11-character video ID and click Load Transcript (ToolerBox key required).
  • Format sends text to OpenRouter; Reset clears input and output.

Output

Action Description
Download Markdown Saves formattr-output.md.
Download Word (.docx) Exports formatted Markdown as a Word document.
Copy Output Copies formatted text to the clipboard.

Word export options

Template Body H1 H2 H3
Default Calibri 18 pt 15 pt 13 pt
Professional Arial 20 pt 16 pt 14 pt
Compact Calibri 16 pt 14 pt 12 pt

Enable Use custom heading styles to override H1/H2/H3 fonts and sizes.

History

Each successful format creates a local history entry. Use Load to restore input and output, or Delete to remove an entry. History is stored in this browser only.

Privacy and BYOK

  • You provide and pay for your own API access.
  • Keys and text go from your browser to OpenRouter when you format.
  • ToolerBox requests use a same-origin proxy that forwards your key but does not store it.
  • No account login; no server-side storage of your content.

Tips and troubleshooting

  • Key required errors — enter a valid OpenRouter key before loading models or formatting.
  • Empty model list — click Load OpenRouter Models again; verify your key.
  • Transcript fails — check ToolerBox key and 11-character video ID.
  • Too much/little editing — change text-change level, preset, or structure modes.
  • Lost presets/history — local storage is per browser; clearing site data removes it.
